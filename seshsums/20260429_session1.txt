--- What happened this session ---

  1. Created axb.py (originally rs232b.py)
     Real-time data logger + live matplotlib plot, reading the same serial source
     as axa.py.

     Logging:
       - Creates results/ if absent.
       - Opens a timestamped .tsv file (YYYYMMDD_HHMMSS.tsv) on startup.
       - Header: timestamp<TAB>value
       - Each row: ISO-format timestamp<TAB>float value
       - Line-buffered so data is safe on Ctrl-C.

     Real-time plot (matplotlib FuncAnimation, 200 ms interval):
       - Raw data in blue, running causal average in red.
       - Interactive controls embedded in the figure (bottom 40%):
           Window slider (10-500): number of samples shown on the X axis.
           Avg slider (1-200): width of the running average.
           Y min / Y max text boxes: type a value and press Enter to rescale
             the Y axis instantly.
           Current time display (lower-right, monospaced, updates each frame).
       - Title bar shows live stats: n points shown, avg width, Y range.
       - Serial reading runs in a daemon thread; plot runs on the main thread.
         Shared data protected by threading.Lock.

     Default starting values are constants at the top of the file:
       DEFAULT_YMIN=0, DEFAULT_YMAX=100, DEFAULT_WINDOW=100, DEFAULT_AVG=20.

  2. Created axanal.py (originally expanalyze.py)
     Reads a results directory containing a .tsv data log and expmodel.txt,
     segments the data per the model, computes per-segment stats, and runs
     Welch's t-tests for any test pairs specified in the model.

     expmodel.txt format (TSV, two sections):
       Model
       <label>  <start_HHMMSS>  <end_HHMMSS>  <cond1>  <cond2>  ...
       ...
       Tests
       <labels_A>  <labels_B>
       ...
       where labels_A and labels_B are comma-joined segment label lists.
       Segment times are HHMMSS on the same calendar date as the data file.
       Boundary matching: nearest sample index to each edge time (no interpolation).

     Output (printed and written to results/expmodel_stats.txt):
       Section 1 — Segment Stats table:
         label, start, end, conditions, n, min, max, mean, sd, actual timestamp range
       Section 2 — T-tests (Welch):
         For each test row, combines all values from left-side labels into one
         group and all values from right-side labels into another, then reports
         n, mean, sd for each group plus t, p, and significance stars.

     Welch's t-test implemented from scratch (no scipy dependency) using the
     regularized incomplete beta function via continued fractions (Numerical
     Recipes algorithm). scipy was installed but its C extensions were broken
     on this machine.

     CLI args:
       --resultsdir DIR   directory to find ax1.tsv and expmodel.txt, and
                          where expmodel_stats.txt is written (default: results/)
       [tsv]              optional positional override for the data file path
       [model]            optional positional override for the model file path
